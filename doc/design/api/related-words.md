# 関連語句一覧（`/related-words`）
## GET
### リクエスト
```
{
  "name": "単語", // 関連語を検索したい語句
  "kana": "タンゴ", // 読み仮名
  "type": 1, // 品詞ID
  "count": 20 // 結果として出力する単語数（未設定の場合20を自動で設定）
}
```

### レスポンス
#### 成功（関連語登録あり）
```
{
  "ok": true, // 取得結果
  "message": "", // メッセージ（成功時は空）
  "words": [ // 関連語リスト
    {
      "id": 1, // 関連語の語句ID
      "name": "文字", // 関連語
      "kana": "モジ", // 関連語の読み仮名
      "type": 1 // 関連語の品詞
    },
    {
      "id": 100,
      "name": "言葉",
      "kana": "コトバ",
      "type": 1
    }
  ],
  "count": 2 // 取得した関連語の数
}
```
#### 成功（関連語登録なし）
```
{
  "ok": true,
  "message": "",
  "words": [],
  "count": 0
}
```
#### 失敗（実行時エラー）
```
{
  "ok": false,
  "message": "E000 エラーが発生しました。"
}
```
